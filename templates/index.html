API گنجور

درباره

این API پایگاه اینترنتی گنجور است، که کمکتون می‌کنه در توسعه نرم‌افزار‌هاتون ازش استفاده کنید به طوری که می‌تونه یه سری شعر رو به صورت رندوم از یه شاعر خاص یا از بین همه‌ی شاعر‌ها بهتون پس بده، خروجی‌ پاسخ داده شما به درخواستی که به سرور می‌فرستید به صورت jsonify شده هست.، برای اطلاعات بیشتر از نحوه‌ی کار API می‌تونید فایل readme رو در گیت‌هاب بخونید.
